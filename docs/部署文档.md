# 部署文档

## 环境要求

| 组件 | 最低版本 | 说明 |
|---|---|---|
| 操作系统 | Windows 10/11 或 Ubuntu 22.04+ | Linux 推荐 |
| Python | 3.12+ | 后端 |
| Node.js | 22+ | 前端构建 |
| PostgreSQL | 16+ | 数据库 |
| Redis | 7+ (或 Windows 上使用 Memurai) | 任务队列 + 缓存 |
| Docker Desktop | 最新版 | 运行 REINVENT4 容器 |
| NVIDIA GPU + Container Toolkit | — | GPU 加速（可选，Vina-GPU 需要） |

## 1. 获取代码

```bash
git clone https://github.com/<用户名>/<仓库名>.git
cd <仓库名>
```

## 2. 安装 PostgreSQL

### Windows
下载安装 [PostgreSQL 16](https://www.enterprisedb.com/downloads/postgres-postgresql-downloads)，安装时设置 postgres 用户密码。

安装后创建数据库：
```sql
CREATE DATABASE reinvent_platform;
```

### Linux
```bash
sudo apt install postgresql-16
sudo -u postgres psql -c "ALTER USER postgres PASSWORD 'your_password';"
sudo -u postgres psql -c "CREATE DATABASE reinvent_platform;"
```

## 3. 安装 Redis

### Windows
下载安装 [Memurai Developer](https://www.memurai.com/get-memurai)（免费版），安装后自动作为 Windows 服务运行。

### Linux
```bash
sudo apt install redis-server
sudo systemctl enable --now redis-server
```

## 4. 安装 Docker 并加载 REINVENT4 镜像

### Windows
安装 [Docker Desktop](https://www.docker.com/products/docker-desktop/)，设置国内镜像加速：
- Settings → Docker Engine → 添加 `"registry-mirrors": ["https://docker.1ms.run"]`

### 加载镜像
```bash
docker load < reinvent4-tools-1.0.0.tar.gz
```

## 5. 配置环境变量

```bash
cp .env.example backend/.env
```

编辑 `backend/.env`：

```ini
DATABASE_URL=postgresql+asyncpg://postgres:<密码>@localhost:5432/reinvent_platform
REDIS_URL=redis://localhost:6379
SECRET_KEY=<随机字符串>
DATA_DIR=./data                    # 数据存储目录
REINVENT4_IMAGE=reinvent4-tools:1.0.0
GPU_ENABLED=true                   # 有 GPU 设为 true，否则 false
ENVIRONMENT=production
```

## 6. 安装依赖

```bash
# 后端
cd backend
pip install -r requirements.txt
python init_db.py

# 前端
cd ../frontend
npm install
```

## 7. 启动服务

需要同时启动 3 个进程：

```bash
# 终端 1: 后端 API
cd backend
uvicorn app.main:app --host 0.0.0.0 --port 8000

# 终端 2: Celery 任务队列
cd backend
celery -A app.workers.celery_app worker --loglevel=info --concurrency=1 --queues=gpu --pool=solo

# 终端 3: 前端开发服务器
cd frontend
npm run dev
```

## 8. 生产环境部署（使用 Nginx）

编译前端：
```bash
cd frontend && npm run build
```

配置 Nginx（参考 `nginx/nginx.conf`），将前端静态文件、API、WebSocket 统一代理到 80 端口。

或使用 Docker Compose 一键启动所有服务：
```bash
docker compose up -d
```

## 9. 访问

打开浏览器：`http://<服务器IP>:5173`（开发模式）或 `http://<服务器IP>`（Nginx 代理模式）

## 故障排查

**任务提交后一直排队**
- 检查 Celery worker 是否启动
- 检查 Redis 是否运行：`redis-cli ping` 应返回 `PONG`

**容器启动失败**
- 检查 Docker 是否运行：`docker ps`
- 检查镜像是否存在：`docker images | grep reinvent4`
- 检查 GPU 驱动：`nvidia-smi`（GPU 模式需要）

**数据库连接失败**
- 检查 PostgreSQL 服务是否启动
- 检查 `backend/.env` 中 `DATABASE_URL` 是否正确
- 检查防火墙是否允许 5432 端口

**页面加载失败**
- 前端开发服务器是否在运行
- 后端 API 是否在运行：访问 `http://localhost:8000/health`
