# 使用文档 / User Guide

## 平台概述

De Nove 双靶点智能分子设计平台是一个基于 AI 的药物分子设计工具，集成了 REINVENT4 强化学习引擎和 DockStream 分子对接框架，支持双靶点协同优化。

---

## 快速开始

### 1. 访问平台

打开浏览器访问 http://localhost:5173

### 2. 创建新项目

1. 点击「新建项目」
2. 输入项目名称和描述
3. 选择目标靶点
4. 配置分子生成参数
5. 点击「提交任务」

### 3. 查看结果

任务完成后，在任务详情页查看：
- 生成的分子列表
- 分数排序
- 2D/3D 分子结构
- 蛋白-配体对接结果

---

## 功能模块

### 1. 项目管理

#### 创建项目
- 输入项目名称和描述
- 选择目标靶点（支持多靶点）
- 配置 REINVENT4 参数：
  - 批次大小 (batch_size)
  - 学习步数 (max_steps)
  - 温度参数 (sigma)
  - 聚合方式 (aggregation)

#### 评分组件配置
- 对接分数权重
- QED（类药性）权重
- SAscore（合成可及性）权重
- 各组分变换函数

### 2. 靶点管理

#### 上传靶点
- 支持 PDBQT 格式文件
- 配置结合位点参数：
  - 盒子中心 (center_x/y/z)
  - 盒子尺寸 (size_x/y/z)
  - 穷举性 (exhaustiveness)

#### PDB ID 自动下载
- 输入 4 位 PDB ID
- 自动从 RCSB PDB 下载
- 自动进行蛋白预处理
- 建议结合位点参数

### 3. 任务执行

#### 提交任务
- 选择项目和参数
- 选择对接后端（Vina/Vina-GPU）
- 提交到 GPU 队列

#### 实时监控
- WebSocket 实时进度推送
- 当前步骤/总步数
- 实时分数更新
- 日志查看

### 4. 结果分析

#### 分子列表
- 按总分排序
- 按各组分分数排序
- SMILES 子结构搜索
- 分数范围过滤
- 步骤范围过滤

#### 分子详情
- 2D 分子结构（RDKit SVG）
- 3D 蛋白-配体复合物（3Dmol.js）
- 评分详情
- 多靶点切换

#### 靶点详情
- 3D 蛋白结构可视化
- 对接盒子显示
- 坐标轴显示
- 参数调整

### 5. 数据导出

#### CSV 导出
- 包含 REINVENT4 完整输出
- Agent/Prior/Target/Score/SMILES
- 各组分原始分数和变换后分数
- UTF-8 BOM 头（Excel 兼容）

## 常见问题

### Q: 任务一直排队中？
A: 检查 Redis 是否运行，Celery Worker 是否启动。

### Q: 对接分数全为 0？
A: 检查 PDBQT 文件是否正确清理（移除 REMARK 等行）。

### Q: 3D 视图无法加载？
A: 检查网络连接，确保能访问后端 API。

### Q: CSV 导出中文乱码？
A: 使用 Excel 的「从文本/CSV」导入功能，选择 UTF-8 编码。
