# 更新日志 / Changelog

## v1.1.0 (2026-05-30)

### 新增功能

#### 蛋白质预处理
- **PDB ID 自动下载**: 从 RCSB PDB 数据库自动下载蛋白结构
- **DockStream 集成**: 使用 DockStream 的 PDBPreparator 进行蛋白修复
- **自动 PDBQT 转换**: PDB → PDBQT 格式转换，支持 OpenBabel
- **盒子参数建议**: 根据参考配体自动计算对接盒子参数

#### 分子 2D 结构预览
- **RDKit 集成**: 使用 RDKit 生成分子 2D 结构 SVG 图片
- **分子列表预览**: 在任务详情页显示分子 2D 结构缩略图
- **详情页大图**: 分子详情页显示完整 2D 结构

#### 3D 可视化增强
- **3Dmol.js 集成**: 替换 NGL Viewer，获得更接近 PyMOL 的效果
- **分子表面显示**: 支持 VDW 分子表面，接近 PyMOL 效果
- **蛋白样式切换**: Cartoon / Surface / Ball+Stick 三种样式
- **配体聚焦**: 自动将配体置于视图中心

#### 靶点详情页
- **3D 蛋白可视化**: 展示蛋白结构和结合位点
- **对接盒子显示**: 绿色线条绘制对接盒子
- **坐标轴显示**: 红(X)/绿(Y)/蓝(Z) 三色坐标轴
- **参数实时预览**: 修改参数时 3D 视图实时更新

#### 任务进度跟踪
- **实时进度显示**: 修复进度显示为 0/1 和 undefined% 的问题
- **WebSocket 推送**: 实时接收任务状态更新

#### 分子列表增强
- **组件分数排序**: 支持按任意评分组件分数排序
- **SMILES 搜索**: 支持 SMILES 子结构搜索
- **分数范围过滤**: 支持按分数范围筛选分子
- **步骤范围过滤**: 支持按生成步骤筛选

### 问题修复

#### PDBQT 文件兼容性
- **移除 REMARK 行**: PDBQT 文件中的 REMARK 行导致 Vina 解析失败
- **清理 Windows 换行符**: 移除 `\r` 字符，避免 Vina 解析错误
- **移除 END 行尾随空格**: END 行的大量尾随空格导致解析失败

#### API 类型兼容
- **SQLite UUID 兼容**: 将所有 UUID 类型参数改为 str 类型
- **修复 500 错误**: 解决 SQLAlchemy text() 与 async session 不兼容问题

#### 文件路径问题
- **绝对路径配置**: 修复 .env 文件依赖工作目录的问题
- **SDF 文件查找**: 支持多种文件名模式（带/不带目标后缀）

#### 数据导出
- **UTF-8 BOM 头**: 在 CSV 文件开头添加 BOM，让 Excel 正确识别编码
- **REINVENT4 原始输出**: 直接返回原始 CSV 文件，包含完整信息

#### 首次加载问题
- **3Dmol.js 加载时序**: 等待脚本加载完成后再初始化 viewer
- **NGL 表面参数**: 使用 `surfaceType: 'av'` 替代默认 SAS

### UI 改进

#### 设计风格
- **紫色渐变主题**: 基于原始设计的紫色渐变配色
- **浅色背景**: 使用 #f8f9fc 浅灰色背景
- **卡片样式**: 16px 圆角，轻微阴影
- **动效**: fadeInUp 入场动画

#### 组件样式
- **表格表头**: 渐变背景，白色文字
- **按钮**: 渐变背景，悬停上浮效果
- **输入框**: 10px 圆角，聚焦时显示紫色边框
- **滚动条**: 紫色主题滚动条

## v1.0.0 (2026-05-28)

### 初始版本

#### 核心功能
- 项目管理（创建/编辑/删除）
- 靶点管理（上传/编辑/删除）
- 任务提交和执行
- 分子列表和详情
- 3D 分子可视化（NGL Viewer）

#### 技术栈
- 前端: React + TypeScript + Vite + Ant Design
- 后端: FastAPI + SQLAlchemy + Redis + Celery
- 容器化: Docker + REINVENT4
