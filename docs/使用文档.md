# 使用文档

## 概述

本平台用于双靶点抑制剂的 AI 分子生成。核心流程：

1. **上传靶点蛋白** — 准备受体 PDBQT 文件和结合位点参数
2. **创建项目** — 选择靶点、配置 AI 生成参数
3. **提交任务** — AI 自动生成并优化分子
4. **查看结果** — 浏览生成的分子、查看 3D 对接构象

## 一、靶点库

### 上传靶点

进入「靶点库」页面，点击「上传靶点」。

| 参数 | 说明 | 示例 |
|---|---|---|
| 靶点名称 | 自定义名称，便于识别 | `EGFR 野生型` |
| 蛋白名称 | PDB ID 或基因名（可选） | `3W32` |
| PDBQT 文件 | 预处理好的受体文件 | `3W32_protein.pdbqt` |
| Center X/Y/Z | 结合位点盒子的中心坐标（Angstrom） | `17.8, 32.5, 12.2` |
| Size X/Y/Z | 盒子的尺寸（Angstrom） | `41, 40, 43` |
| Exhaustiveness | Vina 搜索穷举性（1-128） | `16`（越大越精确但越慢） |

- **结合位点参数**需要从分子对接软件（如 PyMOL、AutoDock Tools）预先测定
- 上传后可编辑或删除

## 二、创建项目

进入「新建项目」，分 5 步完成：

### 步骤 1：项目信息

输入项目名称（如 `EGFR 双靶点抑制剂优化`）和描述。

### 步骤 2：选择靶点

从靶点库中选择本次使用的靶点（双靶点需要至少 2 个）。点击靶点卡片即可选择/取消。

### 步骤 3：REINVENT 参数

| 参数 | 说明 | 推荐值 | 范围 |
|---|---|---|---|
| 最大训练步数 | RL 训练的迭代次数，每步生成一批分子并对接打分 | 200 | 1-500 |
| 每批分子数 | 每轮生成分子数量 | 128 | 16-512 |
| 奖励函数温度 (sigma) | 控制奖励敏感度，越小对高/低分差异越敏感 | 128 | 1-500 |
| 学习率 | 模型参数更新速度 | 1e-4 (标准) | 1e-5 ~ 1e-3 |
| 分数聚合方式 | 多项评分的合并方式 | 几何平均 | 几何平均/算术平均 |
| 分子对接后端 | Vina（CPU）或 Vina-GPU（GPU） | Vina | 取决于硬件 |

- **最大训练步数**：每步约需 20-30 分钟（AutoDock Vina），200 步约需 3-4 天
- **几何平均**：要求所有指标都达标，更严格；**算术平均**：允许短板补偿

### 步骤 4：评分组件

#### 对接打分（自动生成）

每个靶点自动创建一个 DockStream 对接评分组件，可以调整权重。

#### 额外分子属性评分（可选）

| 组件 | 说明 |
|---|---|
| QED 类药性 | 药物相似性定量评估（0-1，越高越好） |
| SAscore 合成可及性 | 合成难度评估（越低越容易合成） |
| 分子量 (MW) | 推荐范围 200-500 Da |
| 脂溶性 (LogP) | 推荐范围 1-3 |
| 极性表面积 (TPSA) | 推荐 < 140 Angstrom² |
| 氢键受体/供体数 | Lipinski 五规则相关 |
| 可旋转键数 | 分子柔性指标 |

**权重**：所有组件的权重之和应为 1.0。点击「自动归一化」可一键调整。（从项目新建任务似乎不可用）

### 步骤 5：确认提交

预览所有配置，确认无误后点击「创建项目并提交任务」。

## 三、任务监控

提交后进入任务详情页：

- **状态**：排队中 → 运行中 → 已完成 / 失败
- **进度条**：当前步数 / 总步数
- **最佳分数**：运行过程中发现的最高分
- **分子列表**：实时显示已生成的分子（待定）

任务运行时间取决于步数和每批分子数。例如 200 步 × 128 分子 ≈ 3-4 天。

## 四、查看结果

### 分子列表

| 列 | 说明 |
|---|---|
| SMILES | 分子的文本表示 |
| 总分 | AI 评分的加权总和 |
| 各组分分数 | 每个靶点的对接打分 + 分子属性原始值 |
| 步骤 | 分子在哪一轮生成 |

- 点击「导出 CSV」可下载全部结果（结果不完善）
- 点击分子旁的「查看」进入 3D 视图

### 3D 蛋白-配体视图

- **蛋白**：二级结构卡通 + 透明表面
- **配体**：元素着色 licorice 表示（只展示最优对接构象）
- **靶点切换**：下拉选择不同靶点，查看该分子在不同蛋白中的对接构象
- **操作**：鼠标左键旋转、滚轮缩放、右键平移

## 五、从已有项目新建任务

在项目详情页点击「新建任务」，可沿用已有项目的靶点配置，只需修改 REINVENT 参数后提交新任务。

## 六、最佳实践

1. **先小规模测试**：首次运行设置 `max_steps=1` 验证配置正确
2. **逐步优化**：根据结果调整评分组件权重
3. **保存 PDBQT**：靶点上传后保留在靶点库，可复用
4. **定期导出**：任务完成后导出 CSV 备份结果
